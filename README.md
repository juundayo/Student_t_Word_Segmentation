# Structure
- `segment_words.py`   | Main word segmentation pipeline.            
- `preprocessing.py`   | Image cleaning and de-slanting.           
-  `distances.py`      | Component gap measurement.                  
-  `student_t.py`      | Mixture model for gap classification.
